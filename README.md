# Life Budget Simulator

人生預算與退休模擬器，一個結合蒙地卡羅模擬、真實市場歷史資料、動態提領策略、資產負債管理與重大人生決策分析的財務沙盤推演工具。

本專案的目標，不只是估算「幾歲可以退休」，而是更貼近真實人生地模擬：

- 收入成長與通膨侵蝕
- 固定支出、變動支出與突發開銷
- 定期定額投資與股票持倉變化
- 房貸、車貸等長期負債
- 退休後提款策略與破產風險
- 買房、買保險、創業等重大決策的長期機會成本

---

## 專案特色

### 1. 深度強化的退休蒙地卡羅模擬

- 以多次隨機模擬評估退休成功率、資產縮水率、財務壓力與可能破產年齡
- 支援薪資年成長率、退休年齡、FIRE 年齡與模擬終止年齡設定
- 內建一般情境與極端壓力情境，便於測試長期抗風險能力

### 2. 真實世界收支與購買力模型

- 支援固定收入、年終月數與每年固定調薪率
- 內建通膨影響，模擬實際購買力下降
- 固定支出可標記為定期定額投資，將現金支出轉為買入股票部位
- 變動支出支援波動範圍，透過蒙地卡羅常態分配模擬不可預期開銷

### 3. 股票資產管理與即時報價

- 支援台股與美股持倉
- 可自動抓取股票報價、歷史年化報酬率、波動率與殖利率
- 台股代碼可自動補 `.TW`，並提供 OTC `.TWO` 後備邏輯
- 自動計算資產現值並以台幣呈現
- 提供資產配置圓餅圖，快速檢視現金與股票比例

### 4. 負債與現金流防護模型

- 可輸入房貸、車貸等分期負債
- 模擬每月還款、剩餘期數與資產負債壓力
- 當現金不足支付開銷時，系統會依持倉比例自動變現股票維持生存
- 可追蹤槓桿倍率與負債資產比，評估整體風險

### 5. GK 護欄動態提領策略

支援 Guyton-Klinger 動態提領邏輯，協助降低退休後過早破產風險：

- 每年 1 月主動提領一年生活費
- 市場下跌時啟動通膨凍結
- 提領率過高時自動削減生活費
- 資產顯著成長時允許增加生活費

### 6. 重大人生決策 Opportunity Cost 分析

- 可模擬保單、買房、買車、創業等重大決策
- 支援一次性大額支出與後續固定現金流負擔
- 系統會同步產生 baseline 對照組
- 可在圖表中直接比較「有做決策」與「沒做決策」的淨資產差異

### 7. 黑天鵝壓力測試與歷史回測

- 提供中位數、樂觀與保守情境分析
- 可測試長期極端熊市或經濟衰退情境
- 支援 Ghost Backtest，將數學模擬結果與真實歷史股價、股息、拆股資料疊加比較
- 內建概念股 Scanner，輔助觀察族群趨勢

## 操作流程

### Step 1. 基本資料與模擬設定

- 填寫目前年齡、退休年齡、模擬終止年齡與 FIRE 年齡
- 設定月薪、年終月數與預期調薪率
- 選擇模擬次數與市場情境
- 建議退休模擬時開啟 GK 護欄動態提領策略

### Step 2. 填寫固定與變動支出

- 新增每月固定支出，例如房租、水電、保費、網路費
- 若為定期定額投資，可勾選該支出為 DCA 並指定買入標的
- 新增變動支出，例如娛樂、醫療、旅遊等
- 為變動支出設定波動範圍，使模擬更貼近真實生活

### Step 3. 建立資產與負債清單

- 輸入現金存款
- 新增股票持倉，支援台股與美股
- 一鍵更新股票報價與投資指標
- 觀察總資產估值與資產配置圓餅圖
- 若有貸款，填入剩餘總額、月付金與剩餘期數

### Step 4. 評估重大財務決策

- 可視需求進入保單、買房、買車或其他決策模組
- 系統會將該決策獨立帶入模擬
- 並與未採取決策的 baseline 結果進行對照

### Step 5. 執行模擬並閱讀報告

完成模擬後，可查看：

- 成功率
- 資產縮水率
- 財務壓力
- 破產年齡
- 資產走勢圖
- 現金流瀑布圖
- 決策前後對照組圖表
- Month-by-Month 明細表
- CSV 匯出資料

## 明細分析亮點

### GK 年度提領追蹤

- 退休後每年 1 月會標記年度提領事件
- 可檢視當年度賣股籌措生活費的細節
- 可觀察護欄是否觸發通膨凍結、生活費削減或增加

### 年度總結

- 彙整每年固定支出、變動支出與貸款支出
- 檢視股息收入與現金流壓力
- 觀察策略在不同年份的運作結果

### CSV 匯出

- 可將逐月資料匯出為 CSV
- 方便使用 Excel 或其他工具做二次分析

## 技術架構

目前專案採用輕量架構：

- 前端：HTML / CSS / Vanilla JavaScript
- 圖表：Chart.js
- 後端：Node.js + Express
- 資料來源：Yahoo Finance API Proxy

伺服器主要用途：

- 提供靜態頁面
- 代理股票報價與歷史資料請求
- 處理台股 `.TW` / `.TWO` 後備查詢
- 提供 Ghost Backtest 所需歷史價格、股息與拆股資料

## 使用情境

這個專案特別適合用來分析：

- 我幾歲可以退休或提早 FIRE
- 目前資產配置是否過於激進或保守
- 若市場長期低迷，我是否還能撐到終老
- 買房會不會拖累退休成功率
- 保單到底是在保風險，還是在犧牲長期複利
- 持續定期定額投入 ETF，對長期淨資產有何幫助

## 注意事項

- 模擬結果僅供財務規劃參考，不構成投資建議
- 歷史資料與市場行為不保證未來報酬
- Yahoo Finance 資料可能存在延遲、缺漏或拆股調整差異，專案已針對部分台股 ETF 額外補強處理
